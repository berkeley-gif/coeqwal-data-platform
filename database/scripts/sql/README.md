# SQL scripts

SQL scripts for database schema management and data operations.

## Usage

**Environment**: AWS Cloud9 (handy for direct RDS PostgreSQL access)

**Connection**:
```bash
psql postgresql://postgres:password@coeqwal-scenario-database-1.clai4yqcyzxh.us-west-2.rds.amazonaws.com:5432/coeqwal_scenario
```

**Running scripts**:
```sql
-- In psql session
\i /path/to/script.sql
```
